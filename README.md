# Algoritmo e Estrutura Dados (AEDS)
Trabalhos feitos para os componentes curriculares Algoritmo e Estrutura de Dados 1, 2 e 3. Essas matérias foram cursadas no Departamento de Ciência da Computação da Universidade Federal de Minas Gerais (UFMG).


### AEDS 1
- Trabalho prático 1 consistia de desenvolver uma calculadora de números complexos sem utilizar biblioteca padrão.
- Trabalho prático 2 consistia de desenvolver uma calculadora de números complexos com entradas e saídas de arquivo sem utilizar biblioteca padrão.

### AEDS 2
- Trabalho prático 1 consistia de implementar um algoritmo para resolver o problema do caixeiro viajante.
- Trabalho prático 2 consistia de desenvolver um jogo de caça palavras.

### AEDS 3
- Trabalho prático 0 consistia de implementar um programa para tratar compartilhamento entre uma rede (grafo).
- Trabalho prático 1 consistia de implementar um programa para ordenação de um arquivo que não cabia na memória principal.
- Trabalho prático 2 consistia de implementar um programa de recomendação de palavras de acordo com a digitação.
- Trabalho prático 3 consistia de desenvolver um programa para simular tempo mínimo necessário para atualização de servidores.
- Resolução de alguns problemas do URI.
